# Spamrito v3.0
Spamrito e um SPAM de alta qualidade, utilizando API de serviços grandes como **Uber**, **99**, **Aiqfome**, **RecargaMulti**, **Vivo** e **Claro**.

## Requisitos
* NodeJS Instalado

## Modulos NPM
* `npm i colors`
* `npm i crypto`
* `npm i request`
* `npm i readline-sync`
* `npm i cli-progress`
