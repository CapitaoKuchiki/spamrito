# Spamrito v3.0
Spamrito e um SPAM de alta qualidade, utilizando API de serviços grandes como Uber, 99, Aiqfome, RecargaMulti, Vivo e Claro.

## Instalação

Você precisar de alguns modulo extra do NPM são eles:&nbsp;
`npm i colors`
`npm i crypto`
`npm i request`
`npm i readline-sync`
`npm i cli-progress`

## Exemplo de uso

Start: ```node spam.js```
